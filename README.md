## Installation

Clone the repository

```bash
git clone https://github.com/krizh4/tourist_site.git
```
```bash
cd tourist_site
```
Install requirements
```bash
pip install -r requirements.txt
```
### > Run on localhost

```bash
cd server
python manage.py runserver
```
